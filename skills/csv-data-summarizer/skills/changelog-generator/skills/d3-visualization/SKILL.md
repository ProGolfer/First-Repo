# D3.js Visualization

Creates D3.js charts and interactive data visualizations.

## When to Use

- When user needs data visualizations
- When interactive charts are required
- When custom visualization logic is needed
- When presenting data analysis results

## Instructions

1. Understand the data and visualization requirements
2. Choose appropriate D3.js chart type:
   - Bar charts for comparisons
   - Line charts for trends
   - Pie/donut charts for proportions
   - Scatter plots for correlations
   - Heatmaps for density
   - Custom visualizations for specific needs
3. Write clean, modular D3.js code
4. Include proper margins, axes, labels, and legends
5. Add interactivity (tooltips, zoom, pan) when useful
6. Ensure responsive design
7. Output as HTML artifact or standalone file

## Best Practices

- Use D3.js v7 or later
- Follow consistent color schemes
- Include proper accessibility attributes
- Add transitions for smooth updates
- Optimize for performance with large datasets
